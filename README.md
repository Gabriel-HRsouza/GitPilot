# 🚀 GitPilot

GitPilot is a desktop application developed in **C# (.NET 8 Windows Forms)** that simplifies Git and GitHub management through an intuitive graphical interface.

The application allows developers to manage repositories, create commits, switch between multiple GitHub accounts, connect repositories, and automate common Git operations without relying on terminal commands.

## ✨ Features

* 📂 Select project folders
* 🔧 Initialize Git repositories
* 👤 Manage multiple GitHub accounts
* 🔄 Switch between accounts quickly
* 📝 Create commits through the interface
* 🚀 Push changes to GitHub
* 📊 View modified files
* 📋 Detailed operation logs
* 🌐 Create public or private repositories
* 🔗 Connect existing repositories
* 🎨 Modern red, purple and black theme

## 🛠️ Technologies

* C#
* .NET 8
* Windows Forms
* Git
* GitHub API

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Gabriel-HRsouza/GitPilot.git
```

Enter the project folder:

```bash
cd GitPilot
```

Open the solution in Visual Studio 2022 or later:

```text
GitPilot.csproj
```

## ▶️ Running the Application

Build and run the project:

```text
F5
```

or

```text
Ctrl + F5
```

## 📋 Requirements

Before running GitPilot, make sure you have installed:

* Visual Studio 2022+
* .NET 8 SDK
* Git

Check Git installation:

```bash
git --version
```
